# 🍔 Online Food Ordering Application

This is a **responsive front-end web application** built as part of the **Simplilearn Full Stack MERN Developer Program**. It allows users to browse food items, add them to a cart, and simulate a payment process using a dummy payment gateway.

## 🚀 Features

- 🧾 Browse food menu items (Burgers, Pizza, Drinks)
- 🧮 Add to Cart and calculate total amount
- 🎯 Category filter (All, Burger, Pizza, Drinks)
- 🧑‍💼 Login & Register popup modal
- 🌗 Dark mode toggle
- 💳 Dummy payment gateway with form validation
- 📱 Responsive design using CSS Grid and Flexbox
- ✨ Scroll animations using ScrollReveal

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Boxicons (for icons)  
- ScrollReveal.js (for animation)

## 📋 How to Run the Project

1. Clone the repository or download the ZIP
2. Extract and open the folder in your code editor
3. Open `index.html` in your web browser
4. Interact with the website (Add items → View cart → Checkout)
